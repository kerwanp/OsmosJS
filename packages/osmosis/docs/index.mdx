Osmosis is a framework for building documentation websites powered by [AdonisJS](https://adonisjs.com) and [OsmosJS](/docs/osmos).

> [!NOTE]
> Osmosis tries to stick as close as possible to the [Github Flavored Markdown](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax) to keep your content readable directly on Github.
